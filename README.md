# Working-with-concurrency-in-golang
# waitgrps, channel, mutex, test cases and race conditions

- To test cases code 
```
go test -v
```

- BenchMark test
```
go test -bench=.
```

- Code coverage result
```
go test -coverprofile=coverage.out
```

- Html code coverage preview
```
go tool cover -html=coverage.out
```
